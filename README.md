# LyricBubble

LyricBubble is a simple, user-driven lyric voting site that surfaces favourite lines from songs. Users vote for the lyric they like most; votes are limited and fade over time so the charts stay current.

**Features:**

- Vote on individual lyric lines; votes decay over time
- Top 10 homepage with deeper chart views and historical charts
- Search by song title or artist; pick lyrics from existing charts
- Minimal user data collection and low hosting costs

**Quick Start**

1. Clone the repository.
2. Open [index.html](index.html) in your browser to view the site.

**Project structure (high level):**

- `index.html` — main entry
- `about/index.html` — about page
- `assets/css/styles.css` — styles

**Development notes**

- Static site: no backend required to preview. Open `index.html` locally.
- Site hosted on Cloudflare Pages
- To contribute, open an issue or submit a pull request with changes.
